This is my attempt to visualize some SARS-CoV-2 data with some R code (please pardon the non-optimal layout). German data is obtained from the Robert Koch Institute. Data used from March 15-April 7, 2020 (not updated every day).

For more extensive information about SARS-CoV-2 in Germany in English, see here.



Cumulative number of confirmed SARS-CoV-2 cases in Germany in linear scale since 100th case

Only the top 10 states with the highest number of cases are plotted.



Number of confirmed cases shown in logarithmic scale

For exponential growth, the amount of time cases increase from 100 to 1000 is roughly the same as the amount of time it goes from 1000 to 10000. By showing the numbers in logarithmic plot, we can see that the growth curves are slowly bending downwards, thanks to the social distancing measures.



Cumulative number of cases per 100,000 population

This graph shows the cumulative increase of cases adjusted for the population size of each state. Per capita, Hamburg’s cases are comparable with Bavaria and Baden-Württemburg.

Number of confirmed new cases in Germany by States since March 1, 2020

This plot shows the increase in the number of confirmed new cases each day, each data point is averaged over 5 consecutive days. Note: This is not necessarily reflective in the increase of actual number of cases each day, since the number of tests for the virus has been increasing as the pandemic progressed.



Cumulative number of deaths in log scale

Data shown since the 5th death.
